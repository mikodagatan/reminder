# Goal

Our primary goal is to build a UI/UX-centric reminder application that simplifies how users manage and are reminded of important tasks and events. Unlike comprehensive tools like Google Calendar, which can be complex and underutilized, our application will focus on ease of use and intuitive design, ensuring users can leverage 100% of its features without a steep learning curve.

We aim to take the friction out of setting up reminders. The application will be a Progressive Web App (PWA), leveraging the latest notification capabilities offered by Rails 8 to provide a seamless experience across devices.

## What is this Reminder Application?

At its core, the application will allow users to easily input what they want to be reminded about and when. We want to provide a more engaging and straightforward way for users to interact with their reminders.

A special take would be warranties. We should be able to receive the warranty receipts so that users can make use of the warranties included when buying items.

## Key Features:

*   **Simplified Reminder Creation:** Easy-to-understand forms for quickly setting up reminders.
*   **Multi-Channel Notifications:** Reminders delivered via:
    *   Email
    *   PWA notifications (utilizing Rails 8 capabilities)
    *   In-app notification system
*   **AI-Powered Scheduling:** An intelligent system to define and manage reminder schedules tailored to individual user needs.

This project is driven by the need for a better, more intuitive way to manage reminders, ensuring users stay on top of their important commitments effortlessly.
