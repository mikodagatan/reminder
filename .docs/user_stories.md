# User Stories

This document outlines user stories for the Reminder Application, categorized by user type or feature set.

## As a New User, I want to:
- `US001:` Sign up for an account easily using my email and a password so that I can start using the application.
- `US002:` Sign up or log in using my Google account so that I don't have to remember another password.
- `US003:` Understand the app's main purpose and features quickly upon my first visit.

## As a Registered User, I want to:
- `US004:` Create a new reminder with a title, due date, and time so that I can be reminded of important tasks.
- `US005:` Add an optional description to my reminder to provide more context for myself.
- `US006:` Set a reminder to recur (e.g., daily, weekly, specific days) so that I don't have to create it multiple times.
- `US007:` See all my upcoming reminders in a clear list so that I know what I need to do.
- `US008:` Edit an existing reminder if details change.
- `US009:` Delete a reminder that is no longer needed.
- `US010:` Mark a reminder as "complete" so it's removed from my active list.
- `US011:` Snooze a reminder for a short period if I can't attend to it immediately.
- `US012:` Receive an email notification before a reminder is due so I don't forget.
- `US013:` Receive a PWA push notification on my device before a reminder is due, even if the app isn't open.
- `US014:` See in-app notifications for my reminders.
- `US015:` Manage my notification preferences (e.g., turn off email or PWA notifications, change default reminder times).
- `US016:` Update my profile information (name, email).
- `US017:` Reset my password if I forget it.

## As a User interested in AI Scheduling, I want to:
- `US018:` Receive intelligent suggestions for when to schedule a reminder based on its type or my habits.
- `US019:` Have the app learn my patterns and suggest better reminder timings over time.
- `US020:` (Optional) Allow the AI to automatically adjust reminder times if I frequently miss or snooze them.

## As a User interested in Warranty Tracking, I want to:
- `US021:` Add a new warranty by inputting product name, purchase date, and expiry date.
- `US022:` Upload a digital copy (photo or PDF) of my warranty receipt or document.
- `US023:` Receive a reminder before a warranty is about to expire so I can take action if needed.
- `US024:` View a list of all my saved warranties and their expiry dates.
- `US025:` Easily find and view the uploaded document for a specific warranty.

## As an Application Administrator/Developer, I want to:
- `US026:` Have a robust logging system to monitor application health and errors.
- `US027:` Easily deploy updates to the application.
- `US028:` Ensure user data is stored securely.

These user stories will help guide the development process and ensure features meet user needs.
