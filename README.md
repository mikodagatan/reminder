# README

## Reminder Application

This application is a user-centric reminder tool designed for simplicity and ease of use. It aims to provide a straightforward way for users to set and receive reminders for important tasks and events, without the complexity of more feature-heavy calendar applications.

The application is a Progressive Web App (PWA) built with Rails 8, leveraging its notification capabilities for a seamless cross-device experience.

### Key Features:

*   **Simplified Reminder Creation:** Easy-to-understand forms for quickly setting up reminders.
*   **Multi-Channel Notifications:** Reminders delivered via:
    *   Email
    *   PWA notifications (utilizing Rails 8 capabilities)
    *   In-app notification system
*   **AI-Powered Scheduling:** An intelligent system to define and manage reminder schedules tailored to individual user needs.
*   **Warranty Tracking:** A feature to upload and manage warranty receipts, ensuring users can easily access them when needed.
